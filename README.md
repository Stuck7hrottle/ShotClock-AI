# 🎯 ShotClock AI: Rate of Fire & Split Analyzer

ShotClock AI is a Python-based tool designed to analyze the Rate of Fire (ROF) and shot-to-shot "splits" from video recordings of firearms. 

By utilizing **Acoustic Onset Detection**, the system identifies the high-pressure "cracks" of gunshots in an audio stream, allowing for millisecond-level precision that exceeds standard video frame rates.

## 🚀 Features
* **Video-to-Data:** Upload `.mp4` or `.mov` files to extract shot data.
* **Split Tables:** View precise time intervals between every shot in a string.
* **Visual Analytics:** Real-time graphing of shot consistency.
* **CLI Support:** Batch process files via terminal for power users.
